# BoilerBooks---API-Frontend-

## RUN API

DO NOT FORGET TO CREATE A .ENV !!
Exemple : 
```
DB_HOST=localhost
DB_USER=user
DB_PASSWORD=password
DB_NAME=postgres
DB_PORT=5432
SECRET_KEY=secret
```

```sh
cd api
npm i 
npx nodemon index.js
```

## RUN FRONTEND
```sh
cd frontend
npm i 
npm run dev
```
